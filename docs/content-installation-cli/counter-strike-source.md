# Counter-Strike: Source

## Prerequisites

- Source SDK Base 2006 / 2007
- [Counter-Strike: Source](https://store.steampowered.com/app/240/CounterStrike_Source/)

## Base content

### Install

```powershell
.\CSS.ps1 -Base
```

### Uninstall

```powershell
.\CSS.ps1 -Base -Uninstall
```

## Maps content

### Install

```powershell
.\CSS.ps1 -Maps
```

### Uninstall

```powershell
.\CSS.ps1 -Maps -Uninstall
```
