# Half-Life Deathmatch: Source

## Prerequisites

- Source SDK Base 2006 / 2007
- [Half-Life Deathmatch: Source](https://store.steampowered.com/app/360/HalfLife_Deathmatch_Source/)

## Base content

### Install

```powershell
.\HL1MP.ps1 -Base
```

### Uninstall

```powershell
.\HL1MP.ps1 -Base -Uninstall
```

## HL1 content

### Install
